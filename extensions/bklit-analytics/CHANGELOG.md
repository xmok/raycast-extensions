# Bklit Analytics Extension Changelog

## [Initial Version] - 2026-01-22

### Added

- **Analytics Snapshot Command**: View comprehensive website analytics directly in Raycast with visual charts
  - Top Countries: See visitor distribution by country with flag indicators and unique visitor counts
  - Device Usage: Monitor mobile vs desktop traffic with percentage breakdowns
  - Top Referrers: Track traffic sources with view counts and percentages
  - Top Pages: Identify most popular pages on your site
- **Menu Bar Integration**: Quick access to top country analytics right from your menu bar
  - Displays flag emoji of your top traffic country
  - Shows visitor count at a glance
  - Refreshes automatically every 5 minutes
- **Secure Authentication**: API key and project ID configuration through encrypted Raycast preferences
- **Custom Dashboard Support**: Configure custom Bklit dashboard URLs for self-hosted instances
- **Visual Charts**: Beautiful bar and doughnut charts powered by QuickChart for data visualization
- **Smart Caching**: Efficient data fetching with caching to minimize API calls
