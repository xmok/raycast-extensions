# Raycast Weekly Newsletter

Browse and interact with the [Raycast Weekly](https://raycastweekly.substack.com/) Newsletter directly from Raycast.

## Features

- **Browse Posts**: Explore and read the latest newsletter issues with real-time stats like likes, comments, and reading time.
- **In-Depth Reading**: Read full posts in a clean, markdown-rendered view within Raycast.
- **Subscribe**: Subscribe to the Raycast Weekly newsletter!
- **Submit Content**: Easily share your content ideas, feedback, or suggestions through a built-in submission form. (currently via Mailto)
