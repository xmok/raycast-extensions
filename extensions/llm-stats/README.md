# LLM Stats for Raycast

The largest collection of high-quality evaluations for every AI modality—language, vision, code, reasoning, and beyond

Based on the [llm-stats.com](https://llm-stats.com).

# Features

- View LLM Leaderboards by Arenas (Chat Arena, Text to SVG, etc.)
- View LLM Benchmarks (SWE, Aider, MMLU, etc.)
- Sort LLM Models by criteria (context size, throughput speed, input/output price, parameters count, etc.)
- Compare LLM Models with other models
- Open LLM Playground
