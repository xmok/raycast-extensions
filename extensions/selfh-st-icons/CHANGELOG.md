## [1.0.0] - 2025-05-09

### Added
- Initial release of the Selfh.st Icons Raycast extension
- Search functionality for finding icons in the selfh.st collection
- Grid-based icon preview interface
- Support for multiple icon formats (PNG, SVG, WebP)
- Light and dark variant support for compatible icons
- Local caching system with 24-hour refresh
- Download functionality to save icons locally
- Copy URL actions for different formats and variants
- Error handling and user feedback
- Performance optimizations for search and display
