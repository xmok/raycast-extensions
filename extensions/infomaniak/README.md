# Infomaniak

This is a [Raycast](https://raycast.com/) extension for [Infomaniak](https://www.infomaniak.com/) - _Sovereign Cloud Service Provider_.

- User Management

## ‚öôÔ∏è Configuration

- **API Token**:

    1. `Navigate` to [API tokens](https://developer.infomaniak.com/getting-started#:~:text=More%20info-,Authentication%20and%20Authorisation,-We%20use%20OAuth2)
    2. `Click` on **Create a token**
    3. `Select` either **All** for scopes or get the following: `accounts`, `drive`
    4. `Click` on **Create the token**

- **Account ID**: The easiest way is to copy this from the URL

    1. `Log In` to **Infomaniak**
    2. `Copy` the first "number" in the bar
