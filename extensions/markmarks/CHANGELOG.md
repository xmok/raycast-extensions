# MarkMarks Changelog

## [1.1.0] - 2026-01-20

- Add Dia browser support

## [Initial Release] - 2026-01-08

- Add bookmark from Safari, Chrome, or Arc browser tabs
- List and search bookmarks with hierarchical group display
- Display website favicons for each bookmark
- Edit, delete, and move bookmarks between groups
- Support for ungrouped bookmarks at root level
- Markdown file persistence for portability
