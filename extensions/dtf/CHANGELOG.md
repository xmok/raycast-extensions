# DTF Changelog

## [Initial Release] - 2025-12-19

### Features

- Browse latest posts from DTF feed with sorting options
- View popular and trending posts by time period (Today, Week, Month, All Time)
- Read editorial news from DTF team
- Search posts across all DTF content
- Browse posts by topics/categories (Games, Movies & TV, Anime, etc.)
- Explore top user blogs with rankings and activity stats
- Menu Bar command with extensive customization
- Full post preview with rich content rendering
- AI-powered features: Summarize, Translate, Key Points, Quick TLDR
- AI tools via `@dtf` mentions for querying DTF in any AI chat

### Menu Bar (macOS only)

- Multiple display modes: icon, post count, title, stats
- Configurable sections: News, Popular, Latest, Topics, Top Blogs
- Post display customization: views, comments, time, authors, categories
- Title rotation between posts
- Quick actions: Search, Open DTF, Refresh

### AI Features

- Summarize posts in 2-3 paragraphs
- Translate to English
- Extract key points as bullet list
- Quick TLDR copied to clipboard
