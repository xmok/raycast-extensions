# Query.Domains Changelog

## [Initial Version] - 2026-01-24

### Features

- **Bulk Domain Check**: Check domain availability across multiple TLDs simultaneously
- **Custom TLD Lists**: Configure your preferred TLDs in extension preferences
- **Smart TLD Detection**: Enter full domain names (e.g., `example.xyz`) to check specific TLDs alongside defaults
- **WHOIS Information**: View registration and expiration dates for registered domains
- **Traffic Analytics**: See monthly visitor statistics for registered domains
- **Quick Actions**: Copy domains, visit websites, and manage settings with keyboard shortcuts
- **User Query Priority**: User-specified domains always appear first in results
- **Categorized Results**: Separate sections for available and registered domains
