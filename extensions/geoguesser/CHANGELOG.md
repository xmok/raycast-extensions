# Geoguesser Changelog

## [Initial Version] - 2025-12-19

### Features Added

- **Profile Viewer** - View stats, level, XP, streaks, and game history
- **Daily Challenge** - See today's challenge, leaderboard, and your score
- **Recent Games** - Browse game history (Daily Challenges, Streaks, Duels)
- **Game Details** - Round-by-round breakdown with scores, distances, and map links
- **Duel Details** - Team stats, health tracking, rating changes, and match results
- **Plonk It Integration** - Quick links to practice any country (Cmd+1-9 shortcuts)
- **TypeScript** - Full type safety with comprehensive interfaces
- **Authentication** - Secure \_ncfa token support (1-year validity)
