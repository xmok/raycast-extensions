# Addy

Manage your [Addy.io](https://addy.io) email aliases directly from Raycast. Create, search, and send emails using your anonymous aliases without leaving your keyboard.

## Features

### Generate Alias
Quickly generate a new random email alias with your default settings.

### Create Custom Alias
Create a custom email alias with full control over:
- Local part (the part before @)
- Domain selection
- Alias format
- Description
- Recipients

### My Aliases
View, search, and manage all your existing email aliases:
- Search through your aliases
- View alias details (forwards, replies, description)
- Toggle aliases on/off
- Edit alias descriptions
- Copy alias email to clipboard
- Delete or forget aliases

### Send an Email from an Alias
Compose sender addresses for your Addy aliases to send emails anonymously:
- Select from your existing aliases
- Enter recipient email (auto-detects from selected text)
- Generates properly formatted sender address
- Copies to clipboard for use in your email client
