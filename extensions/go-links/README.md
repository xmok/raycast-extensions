# Go Links

Search and open [Tailscale go links](https://tailscale.com/kb/1100/golinks) directly from Raycast.

## Requirements

- You must be connected to a Tailscale network with the [golink service](https://github.com/tailscale/golink) running
- The `http://go/` shortcut must be accessible from your machine

## Usage

1. Open Raycast and search for "Go Links"
2. Type to filter through your available go links
3. Press Enter to open the selected link in your browser
