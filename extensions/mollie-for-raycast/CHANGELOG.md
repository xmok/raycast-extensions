# Mollie Changelog

## [Initial Release] - 2026-01-24

First launch of Mollie for Raycast.

### Features

- **Menu Bar Revenue**: Display daily sales revenue and transaction count in the menu bar
- **View Payments**: Browse and filter payments by status, refund payments, open in dashboard
- **View Subscriptions**: List all subscriptions across customers with status filtering, cancel subscriptions
- **Create Payment Link**: Generate payment links with custom amounts, descriptions, and expiry dates
