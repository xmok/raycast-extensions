# Contributing to Table Converter

Thank you for your interest in contributing to the Table Converter Raycast extension!

## How to Contribute

### Reporting Bugs
- Open an issue in this repository.
- Describe the bug clearly.
- Provide steps to reproduce, including the table or markdown snippet that caused the issue.

### Suggesting Enhancements
- Open an issue and tag it as an `enhancement`.
- Explain why this feature would be useful.

### Pull Requests
1. Fork the repository.
2. Create a new branch: `git checkout -b my-feature-branch`.
3. Make your changes and commit them: `git commit -m 'Add some feature'`.
4. Push to the branch: `git push origin my-feature-branch`.
5. Submit a pull request.

## Development Setup

1. Clone the repo.
2. Run `npm install`.
3. Run `npm run dev` to start the Raycast development server.

## Code Style
- We use `Prettier` and `ESLint`.
- Run `npm run lint` and `npm run fix-lint` before committing.
