# Simple Dictionary

A simple Raycast extension to look up words and save definitions to a list of favorites.

## ✍️ Language

When searching, you can select a language from the list of available languages. If no language is selected, the extension will use the default language set in the command preferences.

If neither a language is selected nor a default language is set, the extension will show an Error View prompting you to do so.
