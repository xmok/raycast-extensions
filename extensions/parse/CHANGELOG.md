# Parse Changelog

## [1.1.0] - 2026-01-20

- Added Windows support
- Updated dependencies
- Fixed infinite loop issue caused by `useEffect` + `setState`

## [1.0.0] - 2026-01-16

- Speed read text at 300-700 WPM using RSVP technique
- Automatic clipboard integration with history
- Visual ORP (Optimal Recognition Point) highlighting
- Progress bar tracking
- Adjustable reading speed on the fly
- Play/pause controls with keyboard shortcuts
