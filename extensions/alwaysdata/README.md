# alwaysdata

This is a Raycast extension for [alwaysdata](https://alwaysdata.com/) - _The European Cloud_.

## ðŸš€ Getting Started

1. **Install extension**: Click the `Install Extension` button in the top right of [this page](https://www.raycast.com/xmok/alwaysdata) OR `install` via Raycast Store

    <a title="Install alwaysdata Raycast Extension" href="https://www.raycast.com/xmok/alwaysdata"><img src="https://www.raycast.com/xmok/alwaysdata/install_button@2x.png?v=1.1" height="64" alt="" style="height: 64px;"></a>

2. **Get your alwaysdata API Key**:

    - `Navigate` to [TOKENS](https://admin.alwaysdata.com/token/)
    - `Click` "Generate a token"
    - `Enter` "Application" e.g. `raycast-alwaysdata`
    - `Click` "Submit"
    - `Copy` the token and `Paste` in Preferences or at first prompt
