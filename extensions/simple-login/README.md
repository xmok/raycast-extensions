# SimpleLogin

Create and manage email aliases with [SimpleLogin](https://simplelogin.io) to protect your inbox from spam and trackers.

## Setup

This extension requires a SimpleLogin API key to authenticate with your account.

### Getting your API Key

1. Go to [app.simplelogin.io/dashboard/api_key](https://app.simplelogin.io/dashboard/api_key) and log in if prompted
2. Under **New API Key** input a desired name (e.g. Raycast) and click **Create**
3. Copy the generated API key
4. Paste the API key into the extension preferences when prompted
