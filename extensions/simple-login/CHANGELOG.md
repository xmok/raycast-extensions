# Simple Login Changelog

## [Initial Version] - 2026-01-21

- Added "Create Random Alias" command to quickly generate a random email alias and copy it to clipboard
- Added "Create Custom Alias" command to create aliases with a custom prefix and domain selection
- Added "My Aliases" command to view, search, and manage all your email aliases
- Added ability to enable/disable aliases
- Added ability to pin/unpin aliases for quick access
