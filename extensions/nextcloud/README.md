# Nextcloud

This is a [Raycast](https://raycast.com/) extension for [Nextcloud](https://nextcloud.com/).

## ⚙️ Configuration

- Go to `Settings` > `Security` > `Devices & sessions` > `Create new app password` to obtain an app password for the extension.
- You can also set a custom username for files owner in Search i.e. if your Nextcloud username is different from the files owner username e.g. in education/corporate instances.
